## 这仓库是干嘛的？

最近想开展洋葱星球学习机破解项目

然后发现：
- 官方固件有点屎
- 想装点东西装不了
- 想删点东西删不掉

所以就开了这个仓库，**主要折腾三件事**：

1. **搓TWRP** —— 得先有个能用的Recovery吧
2. **搞Root** —— 没Root玩个锤子
3. **备份/定制固件** —— 怕变砖，也怕官方哪天破脸（bushi

## 目前有啥

```
.
├── LICENSE               # AGPLv3，爱怎么改怎么改，改完记得开源
├── README.md            # 就这个
└── S30/
    └── 固件/
        ├── boot_a.img     # S30的boot
        └── vendor_a.img   # 这个走了LFS，clone完记得 git lfs pull
```

`vendor_a.img` 太特么大力，GitHub不让传大文件，**所以用LFS硬塞进去了**。  
如果你 `git clone` 下来发现这文件是空的，说明没装LFS，或者没pull。
